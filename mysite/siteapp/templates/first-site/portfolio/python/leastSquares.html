This tutorial involves using least squares to find best fit functions to a $sin$ plot with some noise added. To start, we will fit using a $\textit{Standard Basis Function: }c_0+c_1x+c_2x^2+...+c_nx^n$. For this example, we will use the first four terms, so that we may let matrix $A$ have column space $x^0,x^1,x^2,x^3$ and coefficient vector $\vec{c}=<a,b,c,d>$. Because the vector $\vec{c}$ has constants for elements, I've named them $a,b,c$ and $d$. The least squares fit for this standard cubic basis function is the projection of our data into the column space (range space) of $A$, which can be computed by solving $A^TA\vec{c}=A^T\vec{y}$. Below is an example of the solution to this linear system:

There are library functions for bot least squares and standard basis polynomials in numpy, linalg.lstsq(A,ys)[0] returns a 1D numpy array just like the one above. For standard basis function, there is numpy's poly1d(coeffs), which takes an array of coefficients starting with $c_0$ and returns standard basis function with those coefficients. This function returns an array of y values for an array of discretely sampled x values:

#define matrix A with standard basis range space and dot it with x vector
A = np.vstack([xs**3, xs**2, xs**1, xs**0]).transpose()
coeffs = np.linalg.lstsq(A, ys)[0]#extract coeffs from lstq's poly1d output

Next, we were to assume a $\textit{Chevyshev Basis: }c_0x^0,c_1x^1,c_2\left(2x^2-1\right),c_3\left(4x^3-3x\right)$. A $\textit{Chevyshev Basis}$ can converge more closely using fewer terms because each basis column is orthogonal to all other columns. A $\textit{GramSchmidt}$ process can be used to generate an infinite Chevyshev basis function, a process similar to generating a Fourier series. Another way to minimize error is to use the first derivative of the resulting system to find the minimum of the linear combinations of the matrix $A$. Scipy's minimize function returns a dictionary with an $x$ array containing the vector $\vec{c}$ at the minimum. Below, the function squares returns the sum of squared residuals. This is also known as an objective function:

def squares(ab):#objective function
    a = ab[0]
    b = ab[1]      #extracts initial guess
    c = ab[2]      #
    d = ab[3]
    return sum([((a + b*xs[i]+c*(2*xs[i]**2-1)+d*(4*xs[i]**3-3*xs[i]))-ys[i])**2 
                for i in range(len(xs))])#returns sum of squared residuals
result = minimize(squares,[.3,.3,.2,.1,])#minimizes objective function given initial guess
result#display results
