Linear Regression for Immigration and Extinction rates

Start by letting Jupyter notebook know the plots plots are to be displayed in the browser window, rather than in a separate one with %matplotlib inline. Then import numpy because it has fast linear algebra libraries and intuitive vector syntax, pandas because it has very fast, easy to use data frame classes, and matplotlib.pyplot for plotting. Here, numpy.linalg.lstsq was used to write a small function that decides whether or not to force intersection with the origin based on the number of arguments, which has been denoted linregBioG(*args) below. As it turns out, the machine learning library scikit-learn, has a linear regression tool that can force the intercept, as well as many other tools for model fitting and classification. Here is an example of a classification problem solved using sklearn. But I found that out only after I wrote this.

%matplotlib inline 
import numpy as np #import numerical and linear algebra libraries
from pandas import read_table #import data frame libraries
from matplotlib import pyplot as plt #import plot libraries


def linregBioG(*args):  #Generic least squares linreg that allows to force intercept at origin
   
    xs=args[1]
    if len(args)==2: #if only x and y data
        A=np.vstack([xs,xs**0]).transpose() #create matrix A with column space of standard basis
    elif len(args)==3 and args[2]==0: #if extra arg forcing zero intercept
        A=np.vstack([xs,xs*0]).transpose() #create same matrix A but force x**0 column to all zeroes
    else: #don't be tryin nothin funky
            print('invalid entry, program will crash on params call soon!') 
            return  #lazy error check here, this return will cause an error in the params call later 
    return np.poly1d(np.linalg.lstsq(A,args[0])[0])#lstsq minimizes SSR, then return poly1d

Here I read in the data. I use the Pandas library to read the .csv file in as a data frame. Because my regression function uses Numpy's linear algebra libraries I need to pass Numpy arrays as arguments, so I use data_frame.as_matrix() to convert my data frame columns to Numpy arrays. After reading it in I fit my Parameters using linregBioG(*args).

An Intercept of 6.7056 seems to correspond with the graph above, for future use, I have named it ix . The other coefficient, stored in iParams[1], is the slope of our linear relationship, I have named it $a_1$. We'll use this to extract $P$ using the equation:

$$ a_1 = -\frac{I_x}{P} \Rightarrow P = -\frac{I_x}{a_1} $$

Solving $R_{t+1}=R_t$ is the same as solving $I_t=E_t$ after substituting $R_t$ for $R_{t+1}$ and subtracting $R_t$ from both sides, which leaves us to solve:

$$ I_x + a_1 R_e = a_2R_e \Rightarrow R_e = \frac{I_x}{a_2-a_1} $$
